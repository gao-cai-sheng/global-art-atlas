# Global Art Atlas

Static site build for the `Global Art Atlas` project.

## Files

- `index.html`: GitHub Pages entry point
- `global art atlas.html`: named local edition
- `world art map.html`: compatibility entry
- `world_map_robinson.svg`: world map base layer

## Publish

This repository is ready for GitHub Pages. Once pushed to GitHub, enable Pages from the repository settings and publish from the default branch root.
